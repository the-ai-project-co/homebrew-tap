# Homebrew Tap for Nimbus

AI-powered cloud infrastructure management CLI.

## Installation

```bash
brew tap the-ai-project-co/tap
brew install nimbus
```

## Upgrade

```bash
brew upgrade nimbus
```

## Uninstall

```bash
brew uninstall nimbus
```

## Available Formulas

- **nimbus** - AI-powered cloud infrastructure management CLI
